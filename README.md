#  Application de Collecte et Analyse des Données Bancaires

##  Description du projet

Cette application est un projet de **TP INF232 EC2** consistant à développer une application web de **collecte et analyse des données bancaires**.

Elle permet de simuler le comportement des clients d’une banque à travers la saisie de transactions financières et l’analyse descriptive des données.

---

##  Objectifs

- Collecter les données clients et transactions
- Stocker les données dans le navigateur (localStorage)
- Afficher les données sous forme de tableau
- Réaliser une analyse statistique descriptive
- Générer un graphique des transactions
- Proposer une interface moderne et interactive

---

##  Fonctionnalités

###  Formulaire de saisie
- Nom du client
- Téléphone
- Email
- Ville
- Sexe
- Montant de la transaction
- Type (Dépôt / Retrait)
- Catégorie
- Date

---

###  Affichage des données
- Liste complète des transactions
- Possibilité de suppression des données
- Interface claire et structurée

---

###  Analyse descriptive
- Nombre total de transactions
- Total des dépôts
- Total des retraits
- Moyenne des montants
- Visualisation graphique (bar chart)

---

##  Technologies utilisées

- HTML5
- CSS3 (design moderne bleu/noir/blanc)
- JavaScript (Vanilla JS)
- Chart.js (pour les graphiques)
- localStorage (stockage des données)
